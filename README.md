# Proyecto 1 - Biblioteca de Generación y Manejo de Grafos

## Autor

Ángel Mendoza Hernández

## Descripción

Este proyecto implementa una biblioteca básica orientada a objetos en Python para la generación y manejo de grafos.

Se implementaron las siguientes clases:

- Nodo
- Arista
- Grafo

TAmbién, se implementaron los siguientes modelos de generación de grafos:

- Modelo de Malla
- Modelo de Erdős-Rényi
- Modelo de Gilbert
- Modelo Geográfico
- Modelo Barabási-Albert
- Modelo Dorogovtsev-Mendes

El programa permite exportar los grafos en formato GraphViz (.gv), además de generar imágenes (.png) de los grafos creados.
